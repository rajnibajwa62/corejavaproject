              GIT COMMANDS
1.git config :- sets configure values for your name,email and more.

Example: git config--global user"Your name"
         git config--global user.name"Your email"
2.git init :- initialize a git repository.creat the intial.get directory in a new or in an existing project.
Example:git init
3.git add :-added files changes in your working directory to your index.
Example: git add.
4.git commit :-takes all the changes written in index,create a new commit object pointing to it and sets the branch to point to that new commit.
Example: git commit -m"your comment"
5.git status :show you the status of files in the index versus the working directory. it will list out files that are untracked,modified and staged.
Example: git status
6.git log :showa a listed of commits on a branch including the corresponding details.
Example: git log
7.git push :pushes all the modified local objects to the remote repository and advances its branches.
Example: git push origin master
8.git remote add origin:to add a new remote git repository.
Example: git remote and origin
